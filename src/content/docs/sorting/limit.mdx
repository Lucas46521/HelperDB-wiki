## Caracter√≠sticas Importantes

### ‚ö° **Performance Otimizada**
- Para a busca assim que atinge o limite
- Reduz uso de mem√≥ria
- Ideal para grandes conjuntos de dados

### üî¢ **Valida√ß√£o de Limites**
- Aceita apenas n√∫meros positivos
- Limite 0 retorna array vazio
- Limite maior que total dispon√≠vel retorna todos

### üéØ **Flexibilidade de Uso**
- Funciona com dados globais (key vazio)
- Suporte a arrays espec√≠ficos
- Combina perfeitamente com sort() e skip()

```javascript

  // Obter dados com pagina√ß√£o
```
