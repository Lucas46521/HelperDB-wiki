## 🚀 Visão Geral

Helper.DB é uma **biblioteca de gerenciamento de banco de dados em JavaScript** que permite interagir com dados de forma simples e intuitiva. Ideal para aplicações Node.js.

### ✨ Características

- ✅ **API orientada a objetos** — Simples, intuitiva e moderna
- ⚙️ **Abstrações previsíveis** — Métodos consistentes e reutilizáveis
- ⚡ **Performática** — Otimizada para velocidade
- ✅ **Cobertura completa** — Suporte a todas as operações essenciais de banco de dados

## 📦 Instalação

⚠️ Requer **Node.js versão 16.9.0 ou superior**.

```bash title="Instalar via NPM"
npm install helper.db

yarn add helper.db

pnpm add helper.db

bun add helper.db

🔧 Uso Rápido

const { Database } = require('helper.db');

// Criar uma nova instância do banco de dados
const db = new Database('./meu-banco.db');

// Armazenar dados do usuário
await db.set('usuarios.123456789', {
  nome: 'João Silva',
  nivel: 1,
  moedas: 100
});

// Recuperar dados do usuário
const usuario = await db.get('usuarios.123456789');
console.log(usuario); // { nome: 'João Silva', nivel: 1, moedas: 100 }

// Atualizar propriedades específicas
await db.add('usuarios.123456789.moedas', 50);
await db.set('usuarios.123456789.nivel', 2);

// Trabalhar com arrays
await db.push('usuarios.123456789.inventario', ['espada', 'escudo']);

// Verificar se os dados existem
const temUsuario = await db.has('usuarios.123456789');
console.log(temUsuario); // true
