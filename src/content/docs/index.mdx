## About

Helper.db is a powerful **Node.js** module that allows you to easily interact with databases.

- Object-oriented
- Predictable abstractions
- Performant
- 100% coverage of the database operations

## Installation

**Node.js 16.9.0 or newer is required.**

```bash
npm install helper.db
yarn add helper.db
pnpm add helper.db
bun add helper.db
```

### Optional packages

- **zlib-sync** for WebSocket data compression and inflation (`npm install zlib-sync`)
- **bufferutil** for a much faster WebSocket connection (`npm install bufferutil`)
- **@helper/voice** for interacting with databases (`npm install @helper/voice`)

## Example usage

Install helper.db:

```bash
npm install helper.db
```

Register a slash command against the Discord API:

```javascript
const { Database } = require('helper.db');

// Create a new database instance
const db = new Database('./my-database.db');

// Store user data
await db.set('users.123456789', {
  name: 'John Doe',
  level: 1,
  coins: 100
});

// Retrieve user data
const user = await db.get('users.123456789');
console.log(user); // { name: 'John Doe', level: 1, coins: 100 }

// Update specific properties
await db.add('users.123456789.coins', 50);
await db.set('users.123456789.level', 2);

// Work with arrays
await db.push('users.123456789.inventory', 'sword');
await db.push('users.123456789.inventory', 'shield');

// Check if data exists
const hasUser = await db.has('users.123456789');
console.log(hasUser); // true
```
