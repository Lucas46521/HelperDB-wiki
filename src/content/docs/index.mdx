## Início rápido

Configure o Helper.db em seu projeto Node.js em poucos minutos. Siga os passos abaixo para começar a armazenar e recuperar dados facilmente.

### Pré-requisitos
- Node.js (versão 14 ou superior)
- npm ou yarn instalado

### Instalação

```bash
npm install helper.db
```
  

##Exemplo:
```javascript title="Exemplo Simples"
const { Database } = require('helper.db');
const db = new Database('myapp');

// Armazenar dados
db.set('user.name', 'João');
db.set('user.preferences.theme', 'dark');

// Recuperar dados
console.log(db.get('user.name')); // João
console.log(db.get('user.preferences')); // { theme: 'dark' }
```
