# Introduction

Welcome to **Helper.db** - a powerful Node.js module that makes database interactions simple and intuitive.

## What is Helper.db?

Helper.db is designed to simplify database operations while providing:

- **Object-oriented approach**: Clean, readable code patterns
- **Predictable abstractions**: Consistent API across different operations  
- **High performance**: Optimized for speed and efficiency
- **100% coverage**: Complete database operation support

## Key Features

### Simple Database Operations
```javascript
const { Database } = require('helper.db');
const db = new Database('myDatabase.db');

// Insert data
await db.insert('users', { name: 'John', age: 30 });

// Query data
const users = await db.find('users', { age: { $gt: 25 } });

// Update data
await db.update('users', { name: 'John' }, { age: 31 });
```

### Advanced Querying
```javascript
// Complex queries made simple
const results = await db.query()
  .from('users')
  .where('age', '>', 18)
  .and('status', '=', 'active')
  .orderBy('created_at', 'desc')
  .limit(10)
  .execute();
```

### Built-in Backup System
```javascript
// Automatic backups
const backup = new Backup(db);
await backup.create('backup_20240101.db');
await backup.restore('backup_20240101.db');
```
