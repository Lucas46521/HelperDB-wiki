# Installation

Helper.db is available as an npm package and can be installed using your preferred package manager.

## Requirements

<Badge variant="default">Node.js 16+</Badge>
<Badge variant="default">npm/yarn/pnpm</Badge>

- Node.js version 16 or higher
- npm, yarn, pnpm, or bun package manager

## Package Installation

Install Helper.db using your preferred package manager:

<MultiCodeBlock
  title="Install Helper.db"
  options={[
    { label: "npm", code: "npm install helper.db" },
    { label: "yarn", code: "yarn add helper.db" },
    { label: "pnpm", code: "pnpm add helper.db" },
    { label: "bun", code: "bun add helper.db" }
  ]}
  language="bash"
/>

## TypeScript Support

Helper.db includes built-in TypeScript definitions. No additional @types packages are required.

<MultiCodeBlock
  title="TypeScript Project Setup"
  options={[
    { label: "npm", code: "npm install helper.db\nnpm install -D typescript @types/node" },
    { label: "yarn", code: "yarn add helper.db\nyarn add -D typescript @types/node" },
    { label: "pnpm", code: "pnpm add helper.db\npnpm add -D typescript @types/node" },
    { label: "bun", code: "bun add helper.db\nbun add -D typescript @types/node" }
  ]}
  language="bash"
/>

## Verification

After installation, verify that Helper.db is working correctly:

```javascript title="test.js"
const { Database } = require('helper.db');

const db = new Database('test.db');
console.log('Helper.db installed successfully!');
```

Run the test file:

<MultiCodeBlock
  title="Run Test"
  options={[
    { label: "npm", code: "node test.js" },
    { label: "yarn", code: "yarn node test.js" },
    { label: "pnpm", code: "pnpm node test.js" },
    { label: "bun", code: "bun test.js" }
  ]}
  language="bash"
/>

If you see "Helper.db installed successfully!" in your console, the installation was successful.
