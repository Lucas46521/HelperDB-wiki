# Installation

Helper.db is available as an npm package and can be installed using your preferred package manager.

## Requirements

<Badge variant="default">Node.js 16+</Badge>
<Badge variant="default">npm/yarn/pnpm</Badge>

- Node.js version 16 or higher
- npm, yarn, pnpm, or bun package manager

## Package Installation

Install Helper.db using your preferred package manager:

<Tabs>
  <TabItem label="npm">
  ```bash
  npm install helper.db
  ```
  </TabItem>
  <TabItem label="yarn">
  ```bash
  yarn add helper.db
  ```
  </TabItem>
  <TabItem label="pnpm">
  ```bash
  pnpm add helper.db
  ```
  </TabItem>
  <TabItem label="bun">
  ```bash
  bun add helper.db
  ```
  </TabItem>
</Tabs>

## TypeScript Support

Helper.db includes built-in TypeScript definitions. No additional @types packages are required.

<Tabs>
  <TabItem label="npm">
  ```bash
  npm install helper.db
  npm install -D typescript @types/node
  ```
  </TabItem>
  <TabItem label="yarn">
  ```bash
  yarn add helper.db
  yarn add -D typescript @types/node
  ```
  </TabItem>
  <TabItem label="pnpm">
  ```bash
  pnpm add helper.db
  pnpm add -D typescript @types/node
  ```
  </TabItem>
  <TabItem label="bun">
  ```bash
  bun add helper.db
  bun add -D typescript @types/node
  ```
  </TabItem>
</Tabs>

## Verification

After installation, verify that Helper.db is working correctly:

```javascript title="test.js"
const { Database } = require('helper.db');

const db = new Database('test.db');
console.log('Helper.db installed successfully!');
```

Run the test file:

<Tabs>
  <TabItem label="npm">
  ```bash
  node test.js
  ```
  </TabItem>
  <TabItem label="yarn">
  ```bash
  yarn node test.js
  ```
  </TabItem>
  <TabItem label="pnpm">
  ```bash
  pnpm node test.js
  ```
  </TabItem>
  <TabItem label="bun">
  ```bash
  bun test.js
  ```
  </TabItem>
</Tabs>

If you see "Helper.db installed successfully!" in your console, the installation was successful.
