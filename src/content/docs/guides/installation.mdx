# Installation

Get started with Helper.DB in your Node.js project.

## Prerequisites

Before installing Helper.DB, make sure you have:

- Node.js 16.x or higher
- npm, yarn, or pnpm package manager

## Installation Methods

### Using npm

```bash title="Terminal"
npm install helper.db
```

### Using yarn

```bash title="Terminal"
yarn add helper.db
```

### Using pnpm

```bash title="Terminal"
pnpm add helper.db
```

## Verification

After installation, verify that Helper.DB is working correctly:

```javascript title="test.js"
const { Database } = require('helper.db');

// Create a new database instance
const db = new Database();

// Set a value
db.set('test', 'Hello World');

// Get the value
console.log(db.get('test')); // Output: Hello World
```
