# Installation

Install Helper.db using npm:

<CustomCodeBlock 
  code="npm install helper.db"
  language="bash" 
  title="Terminal"
/>

Or using yarn:

<CustomCodeBlock 
  code="yarn add helper.db"
  language="bash" 
  title="Terminal"
/>

Or using pnpm:

<CustomCodeBlock 
  code="pnpm add helper.db"
  language="bash" 
  title="Terminal"
/>

## Prerequisites

Before installing Helper.db, make sure you have:

- **Node.js** version 14.0 or higher
- **npm** or **yarn** package manager
- A Node.js project initialized with `package.json`

## Common Installation Issues

### Node.js Version Compatibility

If you encounter compatibility issues, ensure you're using Node.js 14 or higher:

```bash
node --version
```

### Permission Issues

On Unix systems, you might need to adjust file permissions:

```bash
chmod 755 ./database-directory
```

### SQLite Dependencies

Helper.db uses SQLite under the hood. If you encounter native module compilation issues:

```bash
npm rebuild
# or
npm install --build-from-source
```
